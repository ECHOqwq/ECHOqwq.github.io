Java学习——一些运算符

发表于 2019-05-01 | 更新于 2019-05-27 | 分类于 Java | 评论数：
  1. 区分&和&&运算符：
    &遇到前面一个表达式是假时后面一个照样运算
    而&&遇到前面一个表达式是假时后面一个就不运算了
  2. 异或运算:
    规则：相同为假（0）相反为真（1）
    应用：利用异或运算加密，即一个数异或同一个数两次，结果还是这个数
    注意：它满足交换律鸭!
  3. 取反运算符（~）：
    规则：0变1 1变0
  4. 移位运算符：
    e.g: 3<<2:把3左移两位 6>>2:把6右移两位
    ‘<<’:多出来的位去掉，空掉的位补0
    实质：左移几位其实就是该数据乘以2的几次方
    ‘>>’:多出的位去掉，原来的最高位是0就用0补空位，原来的最高位是 1就用1补空位
    实质：右移几位其实就是该数据除以2的几次方，注意遵循的是整型的除法，比如6/4=1
    ‘>>>’:这是无符号右移，数据进行右移时，高位出现的空位，无论原来 的高位是什么，都用0补空位
    应用1：
    用最有效率的方式算出2乘以8等于多少？
    方式1：
    System.out.println(2*8);
    实际上在电脑里的运算过程:
         0010
    *1000
   --------
     0000
    0000
   0000
  0010
 ----------
    10000
16(2)=10000
所以2*8=16
但这样效率太低了
    方式2：
    2<<3
    这样效率就提高了很多
    应用2：
    方式1：利用第三方变量c来交换
    方式2：
    a=a+b;
b=a-b;
a=a-b;
    但不推荐使用这种方法，如果两个整数的值过大，会超出int范围，会强制转换，数据会变化
    方式3：
    a=a^b;     //a=3^5
b=a^b;     //b=(3^5)^5,即3异或了两次5，又变回3
a=a^b;     //a=(3^5)^3,即5^3^3，又变回5
