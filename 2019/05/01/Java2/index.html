Java学习——Java入门知识

发表于 2019-05-01 | 更新于 2019-05-27 | 分类于 Java | 评论数：

前言:其实感觉java大多规则真的是以C为基础的，所以零零散散总结一点C里自己不怎么强调或者没接触的知识吧
这里就写一些java的基础(还没涉及到面向对象)
注:这些都是参考b站的视频总结出来的，详见https://www.bilibili.com/video/av29306544/?p=55

  1. Java里的标识符规则采用了驼峰规则，即单词开头的字母大写
  2. 里面的变量分为三类：局部变量、成员变量、静态变量
    a.局部变量
    方法或语句块内部定义的变量
生命周期是从声明位置开始到方法/语句块执行完毕为止
    b.成员变量(实例变量)
    方法外部、类的内部定义的变量
从属于对象，生命周期随对象始终
注意：成员变量会自 动初始化，对于数字类型自动赋为0，bool类型自动赋为false
    c.静态变量(类变量)
    使用static定义
从属于类，生命周期伴随类始终，从类加载到卸载
    e.g:
     public class demo1{
     int a;
     static int c;
     public static void main(String[] args){
        int age;
        int salary;
     }
  }
这里main为一个方法，里面定义的age、salary属于局部变量，a是一个成员变量，c是一个静态变量
  3. 常量：不可更改的量
    e.g:
    final double PI=3.14;
    整型常量的四种表示形式:
    十进制整数:99,-500
八进制整数:015(以0开头)
十六进制整数:0x15(以0x开头或0X开头)
二进制整数:0b11011(以0b或0B开头)
  4. 命名规范：
    a.所有变量、方法、类名需见名知意
b.类成员变量:首字母小写和驼峰原则:monthSalary
c.局部变量:首字母小写和驼峰原则
d.常量:大写字母和下划线:MAX_VALUE
e.类名:首字母大写和驼峰原则:GoodMan
f.方法名:首字母小写和驼峰原则:runRun()
  5. 数据类型分为基本数据类型和引用数据类型
    a.基本数据类型
    数值型(又包括整数类型:byte<1>,short<2>,int<4>,long<8>和浮点类型:float<4>,double<8>)
字符型:char<2>
布尔型:boolean<1位>
    b.引用数据类型
    类
接口
数组
  6. 基本数据类型细化
    byte:-128~127(-2的7次方~2的7次方-1)
    short:-32768~32767(-2的15次方~2的15次方-1)
    int:-2147483648~2147483647(-2的31次方~2的31次方-1)
    long:-2的63次方~2的63次方-1
    注意<1>:
    用long声明时如果变量很长，在结尾要加一个L
用float声明时结尾加一个F
e.g:long a=144444444444444444444L;
    float a=3.14F;
    注意<2>:
    浮点型不精确，不要用于比较
如果非要比较，可以使用BigDecimal类(实现任意精度的浮点运算)
BigDecimal bd1=BigDecimal.valueOf(0.1);
BigDecimal bd2=BigDecimal.valueOf(1.0/10.0);
System.out.println(bd1.equals(bd2));
    注意<3>:
    char类型遇到+会自动转为数字
System.out.println('a'+'b'); //输出195
System.out.println(''+'a'+'b') //输出ab
System.out.println(''+'a'+'\t'+'b') //输出a b
  7. 二元运算符运算规则
    整数运算:
    如果两个操作数有一个为long，结果为long
没有long时，结果为int
如果两个操作数有一个是double，结果为double
只有两个操作数都是float时，结果才为float
    取模规则:
    操作数可为浮点数，但一般使用整数
余数符号和左边的操作数符号相同
7%-3=1    -7%3=-1
  8. 字符串连接符:”+”
    e.g
    String a="3";
int b=4;
int c=5;
System.out.println(a+b); //34
System.out.println(b+c+a); //93 (b先和c相加得到9，遇到字符串3变成93)
注意应该区分char(参见第6点的注意3)
  9. 使用Scanner获取键盘输入(同C的scanf)
    e.g
    import  java.util.Scanner;
public class TestScanner {
     public static void main(String[] args) {
          Scanner  scanner =  new Scanner(System.in);
          System.out.println("请输入名字：");
          String  name =  scanner.nextLine(); //输入名字System.out.println("请输入你的爱好：");
          String  favor =scanner.nextLine();  //输入爱好
          System.out.println("请输入你的年龄：");
          int  age = scanner.nextInt(); //输入年龄
          }
}
  10. 简单介绍Math类
    1.java.lang包中的Math类提供了一些用于数学计算的方法
2.Math.random()该方法用于产生一个0~1的double类型的随机数，不包括1
  int i=(int)(6*Math.random()); //产生[0,5]之间的随机整数 
3.Math.PI是圆周率
4.Math.pow(a,b)是求a的b次方
  11. for循环的一个注意点
    for (int i=0;i<3;i++)
  System.out.println("HAHA");
这里的i出了for循环就不能再使用了，因为是在for循环里被定义的
  12. 带标签的break和continue
    “标签”是指后面跟一个冒号的标识符，例如：“label:”。对Java来说唯一用到标签的地方是在循环语句之前
而在循环之前设置标签的唯一理由是：我们希望在其中嵌套另一个循环，由于break和continue关键字通常只中断当前循环，但若随同标签使用，它们就会中断到存在标签的地方
在 “goto有害”论中，最有问题的就是标签，而非goto， 随着标签在一个程序里数量的增多，产生错误的机会也越来越多
但Java标签不会造成这方面的问题，因为它们的活动场所已被限死，不可通过特别的方式到处传递程序的控制权
    e.g
    public class Test18 {
     public static void main(String args[]) {
          outer: for (int i = 101; i < 150; i++) {
               for (int j = 2; j < i / 2; j++) {
                    if (i % j == 0){
                         continue outer;
                         }
                   }
              System.out.print(i + "  ");
         }
    }
}
  13. 语句块
    语句块(有时叫做复合语句)，是用花括号扩起的任意数量的简单Java语句
块确定了局部变量的作用域,块中的程序代码，作为一个整体，是要被一起执行的
块可以被嵌套在另一个块中，但是不能在两个嵌套的块内声明同名的变量
语句块可以使用外部的变量，而外部不能使用语句块中定义的变量，因为语句块中定义的变量作用域只限于语句块
  14. 字符串
    声明
    <1> String 变量名 = "初始值";
<2> String 变量名 = new String("初始值");
第一种形式使用双引号引住的字符串叫做字面量
第二种使用new运算符声明的字符串叫做字符串对象
第二种形式并不常用，推荐使用第一种形式
    连接
    字符串可以进行加法运算，作用是将两个字符串连接在一起，
也可以将字符串与基本类型变量做加法运算
系统会先将基本类型转换为字符串型后进行连接操作
e.g
String str1 = “Hello” + “ World”; //结果为Hello World
String str2 = str1 + 2000; //结果为Hello World 2000
    比较
    字符串也可以进行是否相等的比较，但不能直接使用“==”运算符
要使用equals()方法进行比较
    e.g
    public static void main(String[] args) {
     String a = "Hello";
     String b = "H";
     String c = b + "ello";
     System.out.println(a.equals(c));
}
注:除了equals()以外还有很多其他API，可以自行查阅
    把基本类型转化为String型
    <1> 通过字符串连接将基本类型转换成String类型
    e.g
    public static void main(String[] args) {
     int a = 10;
     double b = 3.14D;
     boolean c = true;
     String sa = "" + a;
     String sb = "" + b;
     String sc = "" + c;
}
使用 + 将基本类型与字符串类型连接时
java首先自动将基本类型转换成字符串类型
然后才连接到一起
    <2> 通过String.valueOf(基本类型数据)方法将基本类型转换成String类型
    e.g
    public static void main(String[] args) {
     int a = 10;
     double b = 3.14D;
     boolean c = true;
     String sa = String.valueOf(a);
     String sb = String.valueOf(b);
     String sc = String.valueOf(c);
}
使用String类的valueOf()方法可以将基本类型转换成String类型
    把String型转化为基本类型
    使用基本类型的包装类
    e.g
    public static void main(String[] args) {
     String s1="1",s2="true",s3="3.14";
     byte b = Byte.parseByte(s1);
     short s = Short.parseShort(s1);
     int i = Integer.parseInt(s1);
     long l = Long.parseLong(s1);
}
即用方法parse……()来转化
    字符串与char类型的转换可以通过字符串的charAt方法完成
    e.g
    public static void main(String[] args) {
     String str = "Hello World";
     char ch1 = str.charAt(6);
     System.out.println("Hello World中第6位下标是字符"+ch1);
}
out:Hello World中第6位下标是字符W
    怎么获得字符的ASCII码值
    char a = 'c';
System.out.println((int)a);
    怎么获得输入的字符串的ASCII码值
    String a = "hello";
for(int i = 0; i < a.length(); i++){
System.out.println((int)a.charAt(i));
}
