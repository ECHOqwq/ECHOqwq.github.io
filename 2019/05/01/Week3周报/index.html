发表于 2019-05-01 | 分类于 周报 | 评论数：

前言:其实感觉java大多规则真的是以C为基础的，所以零零散散总结一点C里自己不怎么强调或者没接触的知识吧

  1. Java里的标识符规则采用了驼峰规则，即单词开头的字母大写
  2. 里面的变量分为三类：局部变量、成员变量、静态变量
    a.局部变量
    方法或语句块内部定义的变量。
生命周期是从声明位置开始到方法/语句块执行完毕为止。
    b.成员变量(实例变量)
    方法外部、类的内部定义的变量。
从属于对象，生命周期随对象始终。
注意：成员变量会自动初始化，对于数字类型自动赋为0，bool类型自动赋为false
    c.静态变量(类变量)
    使用static定义。
从属于类，生命周期伴随类始终，从类加载到卸载。
    e.g:
     public class demo1{
     int a;
     static int c;
     public static void main(String[] args){
        int age;
        int salary;
     }
  }
这里main为一个方法，里面定义的age、salary属于局部变量，a是一个成员变量，c是一个静态变量
  3. 常量：不可更改的量
    e.g:
    final double PI=3.14;
    整型常量的四种表示形式:
    十进制整数:99,-500
八进制整数:015(以0开头)
十六进制整数:0x15(以0x开头或0X开头)
二进制整数:0b11011(以0b或0B开头)
  4. 命名规范：
    a.所有变量、方法、类名需见名知意
b.类成员变量:首字母小写和驼峰原则:monthSalary
c.局部变量:首字母小写和驼峰原则
d.常量:大写字母和下划线:MAX_VALUE
e.类名:首字母大写和驼峰原则:GoodMan
f.方法名:首字母小写和驼峰原则:runRun()
  5. 数据类型分为基本数据类型和引用数据类型
    a.基本数据类型
    数值型(又包括整数类型:byte<1>,short<2>,int<4>,long<8>和浮点类型:float<4>,double<8>)
字符型:char<2>
布尔型:boolean<1位>
    b.引用数据类型
    类
接口
数组
  6. 基本数据类型细化
    byte:-128~127(-2的7次方~2的7次方-1)
    short:-32768~32767(-2的15次方~2的15次方-1)
    int:-2147483648~2147483647(-2的31次方~2的31次方-1)
    long:-2的63次方~2的63次方-1
    注意<1>:
    用long声明时如果变量很长，在结尾要加一个L
用float声明时结尾加一个F
e.g:long a=144444444444444444444L;
    float a=3.14F;
    注意<2>:
    浮点型不精确，不要用于比较
如果非要比较，可以使用BigDecimal类(实现任意精度的浮点运算)
BigDecimal bd1=BigDecimal.valueOf(0.1);
BigDecimal bd2=BigDecimal.valueOf(1.0/10.0);
System.out.println(bd1.equals(bd2));
