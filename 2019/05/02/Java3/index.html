总述:面向过程(C) 面向对象(Java、C++)
什么是面向过程?
我们首先思考”怎么按步骤实现?”并按步骤对应方法一步一步完成
适合简单、不需要协作的任务
什么是面向对象?
OOA：面向对象分析
OOD：面向对象设计
需要大量协作的任务
面向对象和面向过程的总结:
解决简单问题使用面向过程，解决复杂问题宏观上使用面向对象把握，微观处理上仍然是面向过程

  1. 对象的理解
    对象其实就是一个内存块用来表示一堆相关联的数据
    像一个项目里面有财务人员、技术人员、运维人员一样，一个对象包含了不同类型的数据和各种方法

  2. 对象和类的概念
    类可以看做是一个模板，系统根据类的定义来创建对象

    通过抽象创建了类，某个类的对象即为某个类的实例
一个类里面有属性(成员变量)和方法

    e.g 

    public class Stu {
    //属性（成员变量）
    int id;
    String name;
    int age; 
    Computer comp; //计算机 
    void study(){  //方法
        System.out.println("我正在学习！"+comp.brand);
    }  
    //构造方法，用于创建该类的对象，无参的构造方法可以由系统自动创建
    Stu(){ //名字一定要和类名一模一样
    }
    public static void main(String[] args){
    Stu stu=new Stu(); //建立一个Stu型对象stu
    stu.id=0101010;
    Computer c1=new Computer(); //建立一个Computer型对象c1
    c1.brand="DELL"; //给c1赋值
    stu.comp=c1; //把c1的值赋给stu对象里Computer型的成员comp
    stu.study(); //可以通过.来访问对象的相关属性
    }  //main函数是程序执行的入口，可以放在这个类里面，也可以放在别的类里面
}

class Computer{  //再创建一个Computer型的类
    String brand;
}

    注: 

    要想创建类里的对象要调用main函数在里面创造对象 
类和类之间也可以相互嵌套，如上述的Computer类和Stu类
  3. Java虚拟机的内存分析
    分为栈stack、堆heap、方法区method area
    <1> 栈
    栈描述的是方法执行的内存模型，每个方法被调用都会创建一个栈帧(存储局部变量、操作数、方法出口等)，执行完以后会被关闭删去
它属于线程私有，不能实现线程间的共享
先进后出，后进先出
是一个连续的内存空间
    <2> 堆
    堆用于存储创建(new)好的对象(数组也是对象)，所有线程共享
堆是一个不连续的内存空间，分配灵活，速度慢
    <3> 方法区/静态区
    用来存放程序中永远是不变或唯一的内容(类信息、静态变量、字符串常量等)
所有线程共享，实质上也是堆
  4. 构造方法/构造器
    用于对象的初始化
    要点
    <1> 要用new关键字调用
<2> 构造器虽然有返回值，但不能定义返回值的类型，不能再构造器里使用return返回某个值
注意:可以直接写一个不带返回值的return用于结束该方法
<3> 如果没有定义构造器，编译器会自动定义一个无参的构造函数
<4> 构造器的方法名必须和类名一致
    e.g
    class Point {
    double x, y;
    //构造方法名要和类名一致
    public Point(double _x, double _y) {
        x = _x;
        y = _y;  
    }
    public double getDistance(Point p) {
        return Math.sqrt((x - p.x) * (x - p.x) + (y - p.y) * (y - p.y));
    }
}
public class TestConstructor {
    public static void main(String[] args) {
        Point p = new Point(3.0, 4.0);
        Point origin = new Point(0.0, 0.0);
        System.out.println(p.getDistance(origin));
    }
}
  5. 方法
    方法就是一段用来完成特定功能的代码片段。
方法用于定义该类或该类的实例的行为特征和功能实现。 方法是类和对象行为特征的抽象。方法很类似于面向过程中的函数。面向过程中，函数是最基本单位，整个程序由一个个函数调用组成。面向对象中，整个程序的基本单位是类，方法是从属于类和对象的。
    方法的调用方式：
    对象名.方法名(实参列表)
    方法的详细说明
    <1> 形式参数：在方法声明时用于接收外界传入的数据。
<2> 实参：调用方法时实际传给方法的数据。
<3> 返回值：方法在执行完毕后返还给调用它的环境的数据。
<4> 返回值类型：事先约定的返回值的数据类型，如无返回值，必须显示指定为为void。
    e.g
    public class Test20 {
     /** main方法：程序的入口 */
     public static void main(String[] args) {
          int num1 = 10;
          int num2 = 20;
          //调用求和的方法：将num1与num2的值传给add方法中的n1与n2
          // 求完和后将结果返回，用sum接收结果
          int sum = add(num1, num2);
          System.out.println("sum = " + sum);//输出：sum = 30
          //调用打印的方法：该方法没有返回值
          print();
          }
          /** 求和的方法 */
          public static int add(int n1, int n2) {
               int sum = n1 + n2;
               return sum;//使用return返回计算的结果
          }
          /** 打印的方法 */
          public static void print() {
               System.out.println("HAHA");
          }
}
  6. 方法的重载
         方法的重载是指一个类中可以定义多个方法名相同，但参数不同的方法。
     调用时，会根据不同的参数自动匹配对应的方法。
注意：
     <1> 只有返回值不同，不构成方法的重载
         int a(String str){}  void a(String str){}
     <2> 只有形参的名称不同，不构成方法的重载
         int a(String str){}  int a(String s){}
总:
         public static int add(int n1, int n2) {
              int sum = n1 + n2;
              return sum;
         }
         // 方法名相同，参数个数不同，构成重载
         public static int add(int n1, int n2, int n3) {
              int sum = n1 + n2 + n3;
              return sum;
         }
         // 方法名相同，参数类型不同，构成重载
         public static double add(double n1, int n2) {
              double sum = n1 + n2;
              return sum;
         }
         // 方法名相同，参数顺序不同，构成重载
         public static double add(int n1, double n2) {
              double sum = n1 + n2;
              return sum;
         }
         //编译错误：只有返回值不同，不构成方法的重载
         public static double add(int n1, int n2) {
              double sum = n1 + n2;
              return sum;
         }
         //编译错误：只有参数名称不同，不构成方法的重载
         public static int add(int n2, int n1) {
              double sum = n1 + n2;         
              return sum;
         }
