Java学习——random随机数

发表于 2019-05-05 | 分类于 Java | 评论数：

然后这一篇也很简单地总结一下自己花了一个小时学的随机数

  1. Math类的random
    Math.random()方法语句基础上处理可获得多种类型、或任意范围的随机数
    <1> 生成大于等于0小于等于n的随机数
    int num = (int)(Math.random()*n);
    <2> 生成大于等于m小于m+n的随机数
       int num = m+(int)(Math.random()*n);
经验:要生成大于等于m小于n的随机数的方法
   int num = m+(int)(Math.random()*(m-n));
    <3> 生成a~z字符
    char a = (char)('a'+Math.random()*('z'-'a'+1));
  2. Random类
    Random类在java.util.Random里
    和Scanner一样可以通过实例化一个Random对象来创建一个随机数生成器
    <1> 实例化
    Random ran = new Random();
实例化一个ran对象
    <2> 产生随机数(同Scanner)
    int i = ran.nextInt(100);
产生1~100的一个随机数
若括号里没有数字则没有限制范围
也可用ran.nextDouble等
