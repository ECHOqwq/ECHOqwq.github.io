Java学习——Java三大特征细化

发表于 2019-05-08 | 分类于 Java | 评论数：

今天又看了相关的书籍，稍微再自己梳理了一下Java的继承
首先，子类和父类是继承关系，当子类继承父类的成员变量和方法时，
并不是再开辟一块空间，而是指向父类的该变量/方法，而如果子类调
用父类的方法并对方法进行重载，则会重新开辟一块内存空间放新的方
法，指向新的方法，这时候和父类的原方法就没有关系了
e.g
public class TestExtends {
public static void main(String[] args) {
A a = new A();
B b = new B(200);
b.printa();
}
}
class A{
int x = 2;
public void printa() {
System.out.println(x);
}
}
class B extends A{
int x = 100;
public B(int x) {
this.x = x;
}
}
该例输出的x是2，因为B类并没有重写A类的printa方法
所以还是原来的那个空间，可以理解成只是把它调用出来使用
所以输出的值还是A类的x值
若把上例的class B的代码修改
class B extends A{
int x = 100;
public B(int x){
this.x =x;
}
public void printa(){
System.out.println(x);
}
}
即重载了printa方法，这时就不再输出父类的x了
其次，在子类里并不能继承父类的构造方法，利用super其实还是在父类分配的空间里赋值
e.g
public class TestOverload {
public static void main(String[] args) {
Dog dogs = new Dog(“萨摩耶”,”汪汪”);
dogs.Print();
}
}
class Animals{
String name = “animals”;
String breed = “动物”;
String type = “a”;
public Animals(String type,String breed,String name) {
this.type = type;
this.breed = breed;
this.name = name;
}
}
class Dog extends Animals {
public Dog(String breed, String name) {
super(“狗”, breed, name);
// TODO 自动生成的构造函数存根
}
void Print(){
System.out.println(super.name);
}
}
out:汪汪
为什么不是输出父类的name即animals呢？和上面说的一样，其实是在
父类的空间里赋值，所以其实等于是把父类里的name赋为了汪汪，因此
是有给父类赋值的
e.g
public class TestExtends {
public static void main(String[] args) {
B b = new B(200);
b.printa();
}
}
class A{
int x;
public A(int x) {
this.x = x;
}
public void printa() {
System.out.println(x);
}
}
class B extends A{
int x;
public B(int x) {
super(x);
}
public void printa() {
System.out.println(x);
System.out.println(super.x);
}
}
out : 0 200
这里子类的构造方法没有this，因此子类的x没有赋值，是初始值0
而调用了父类的构造，因此父类的x被赋了200
e.g
public class TestOverload {
public static void main(String[] args) {
Duck ducks = new Duck(“嘎嘎”);
Animals a = new Animals(“动物”,”生物”,”many”);
ducks.introduce();
}
}
class Animals{
String name = “animals”;
String breed = “动物”;
String type = “mam”;
public Animals(String type,String breed,String name) {
this.type = type;
this.breed = breed;
this.name = name;
}
public void introduce() {
System.out.println(“我是”+name+”,我的品种是”+breed+”,我的名字是”+name);
}
}
class Duck extends Animals {
public Duck(String name) {
super(“鸭鸭”,”未知”,name);
}
}
这里的introduce之所以可以输出ducks的name、type和breed
是因为它是父类的一个方法，然后子类的super给父类的name、
type和breed赋了值，父类在此输出
所以总结一下我的理解(不懂是不是对的)
java里的子类继承父类其实类似于C里的函数调用函数，不一样的是
它不会开辟新的空间(哪怕是短暂的)，子类调用父类的某个方法，其
实就只在父类里修改，而子类里面其实没有被赋值等操作
