Java学习——集合框架Map

发表于 2019-05-15 | 更新于 2019-05-27 | 分类于 Java | 评论数：

  1. Map接口
    <1> Map集合和Set、List的区别

    List集合存储的是有顺序的，可以存储重复的数据
Set集合存储的是无序的不允许重复的数据
Map集合存储的是键值对数据

    <2> 键值对

    Map集合内部有两个集合
一个集合用于保存Map中的key（键）
一个集合用于保存Map中的value（值）
其中key和value可以是任意数据类型数据

    <3> Map集合中的HashMap类

    put(Object key,Object value)    加入元素，返回与此key关联的原有的value，不存在则返回null
clear()    从集合中移除所有的元素
containsKey(Object key)    根据key从集合中判断key是否存在
containsValue(Object value)    根据value从集合中判断value是否存在
get(Object key)    根据key返回key对应的值
keySet()    返回Map集合中包含的键集合
remove(Object key)    从集合中删除key对应的元素，返回与key对应的原有value，不存在则返回null
size()    返回集合中的元素的数量

注: Map集合中的常用类有Hashtable、HashMap、Properties

    e.g

import java.util.Iterator;
import java.util.Map;
import java.util.Set;

 }
 public class HashMapDemo1 {
     public static void main(String[] args) {
         // 定义集合对象map
         Map<String, Person> map =new HashMap<String, Person>();
         //创建Person对象
         Person p1 = new Person("001","史进");
         Person p2 = new Person("002","李逵");
         Person p3 = new Person("003","吴用");
         //向map集合中添加元素
         map.put(p1.getCardId(), p1);
         map.put(p2.getCardId(), p2);
         map.put(p3.getCardId(), p3);
         // 通过键集判断集合中是否包含"史进"
         if (map.containsKey(p1.getCardId())) {
             System.out.println("通过键集查找到了史进");
         } else { 
             System.out.println("通过键集没有查找到了史进");
         }
         // 通过值集判断集合中是否包含"史进"
         if (map.containsValue(p1)) {
             System.out.println("通过值集查找到了史进");
         } else {
             System.out.println("通过值集没有查找到了史进");
         }
         // 从map集合中获取一个元素
         Person person =  map.get("001");
         System.out.println("键为001的用户，他的姓名是"+person.getName());
         //迭代Map集合中所有的好汉
         System.out.println("迭代集合中所有的元素");
         StringBuffer buffer =new StringBuffer();
         Set<String> set = map.keySet();
         Iterator<String> iterator = set.iterator();
         while(iterator.hasNext()){
             String key = iterator.next();
             Person person2 = map.get(key);
             buffer.append("键为");
             buffer.append(person2.getCardId());
             buffer.append("的用户，他的姓名是");
             buffer.append(person2.getName());
             System.out.println(buffer.toString());
             buffer.delete(0, buffer.length());
         }
         // 从集合中移除史进
         map.remove(p1.getCardId());
     }

}
