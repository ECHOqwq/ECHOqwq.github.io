Java学习——java.util工具包

发表于 2019-05-21 | 更新于 2019-05-27 | 分类于 Java | 评论数：
  1. 日期类
    <1> java.util.Date类
    构造方法：Date()                按照当前系统时间构造一个Date对象
         Date(long date)       按照给定的时间毫秒值构造一个 Date 对象
主要方法: after(Date when)    测试当前对象表示的时间是否在指定时间之后。
         before(Date when)  测试当前对象表示的时间是否在指定时间之后
         getTime()           返回当前对象对应的时间毫秒值
         setTime(long time) 设置时间
    <2> java.util.Calendar类
    主要方法: AM              指示从午夜到中午之前这段时间的 AM_PM 字段值
         DATE            get 和 set 的字段，指示一个月中的某天
         DAY_OF_MONTH    get 和 set 的字段，指示一个月中的某天
         DAY_OF_WEEK        get 和 set 的字段，指示一个星期中的某天
         DAY_OF_YEAR        get 和 set 的字段，指示当前年中的天数
         HOUR            get 和 set 的字段，指示上午或下午的小时
         HOUR_OF_DAY        get 和 set 的字段，指示一天中的小时
         MINUTE            get 和 set 的字段，指示一小时中的分钟
         MONTH            指示月份的 get 和 set 的字段
         PM                指示从中午到午夜之前这段时间的 AM_PM 字段值
         SECOND            get 和 set 的字段，指示一分钟中的秒
         WEEK_OF_MONTH    get 和 set 的字段，指示当前月中的星期数
         WEEK_OF_YEAR    get 和 set 的字段，指示当前年中的星期数
         YEAR    表示年的 get 和 set 的字段
    <3> e.g 1
    public static void main(String[] args) {
    Calendar calendar = Calendar.getInstance();
    int year = calendar.get(Calendar.YEAR);
    int month = calendar.get(Calendar.MONTH);
    int day = calendar.get(Calendar.DATE);
    int hours = calendar.get(Calendar.HOUR);
    int minutes = calendar.get(Calendar.MINUTE);
    int second = calendar.get(Calendar.SECOND);
    StringBuffer buffer =new StringBuffer();
    buffer.append("现在时间是").append(year).append("年").append(month+1).append("月").append(day).append("日").append(hours).append(":").append(minutes).append(":").append(second).append("秒");
    System.out.printl n(buffer.toString());
}

  2. 格式化
    <1> 定义

    格式化是指将日期或数字按照指定的格式显示的过程

    <2> 日期格式化

    java在java.text包中提供了SimpleDateFormat类用于格式化日期
该类根据格式符将日期转换成指定的格式
常用格式符:
    G    年代标志符
    y    年
    M    月d    日
    h    时，12小时制
    H    时，24小时制
    m    分
    s    秒
    S    毫秒
    E    星期
    D    一年中的第几天
    w    一年中第几个星期
    W    一月中第几个星期
    a    上午/下午
    z    时区

    <3> e.g: 将当前时间格式化成yyyy-MM-dd HH:mm:ss:SSS格式

    public static void main(String[] args) {
    java.util.Date date =new java.util.Date();
    java.text.SimpleDateFormat sdf =new java.text.SimpleDateFormat("yyyy-MM-dd HH:mm:ss:SSS");System.out.println(sdf.format(date));
}

    <4> 数字格式化

    java提供了java.util.Locale类，来表示不同国家和地区
Locale.CHINA    中国地区
Locale.US    美国地区
Locale.FRANCE    法国地区
Locale.CANADA    加拿大地区

NumberFormat 是所有数值格式的抽象基类，提供格式化和解析数值的接口
getInstance()            获取常规数值格式
getNumberInstance()        获取常规数值格式
getIntegerInstance()    获取整数数值格式
getCurrencyInstance ()    获取货币数值格式，格式化后的数据前面会有一个货币符号，比如：“￥”
getPercentInstance()    获取显示百分比的格式，比如：小数 0.53 将显示为 53%

    <5> e.g

    public static void main(String[] args) {
    double mynum1 = 230456789;
    double mynum2 = 0.23;
    NumberFormat nf1 = NumberFormat.getInstance(Locale.CHINA);
    NumberFormat nf2 = NumberFormat.getCurrencyInstance(Locale.CHINA);
    NumberFormat nf3 = NumberFormat.getCurrencyInstance(Locale.US);
    NumberFormat nf4 = NumberFormat.getPercentInstance();
    System.out.println(nf1.format(mynum1));
    System.out.println(nf2.format(mynum1));
    System.out.println(nf3.format(mynum1));
    System.out.println(nf4.format(mynum2));
}
  3. Math类
    <1> 总
    java.lang.Math类中，包含用于执行基本数学运算的方法，如指数、对数、平方根和三角函数
所有方法和常量全部都是静态的
    <2> 主要方法
    abs(T a)        返回 long 值的绝对值
acos(double a)    返回一个值的反余弦；返回的角度范围在 0.0 到 pi 之间
atan(double a)    返回一个值的反正切；返回的角度范围在 -pi/2 到 pi/2 之间
ceil(double a)    返回最小的（最接近负无穷大）double 值，该值大于等于参数，并等于某个整数
cos(double a)    返回角的三角余弦
floor(double a)    返回最大的（最接近正无穷大）double 值，该值小于等于参数，并等于某个整数
log(double a)    返回 double 值的自然对数（底数是 e）
log10(double a)    返回 double 值的底数为 10 的对数
max(T a, T b)    返回两个 double 值中较大的一个
min(T a, T b)    返回两个 long 值中较小的一个
pow(T a, T b)    返回第一个参数的第二个参数次幂的值
random()        返回带正号的 double 值，该值大于等于 0.0 且小于 1.0
round(float a)    返回最接近参数的 int
sin(double a)    返回角的三角正弦
sqrt(double a)    返回正确舍入的 double 值的正平方根
tan(double a)    返回角的三角正切
