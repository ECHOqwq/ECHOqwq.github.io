Java学习——文件

发表于 2019-05-22 | 更新于 2019-05-27 | 分类于 Java | 评论数：
  1. File类
    <1> File类的对象表示磁盘、目录、文件
    <2> 常用方法
    canRead                文件是否可读
canWrite                文件是否可写
delete                删除文件或目录
exists                文件或目录是否存在
getAbsolutePath        获取绝对路径
getFreeSpace            获取分区的剩余空间
getTotalSpace        获取分区的总空间
getUsableSpace        获取分区的已用空间
getName                获取文件或目录的名称
isDirectory            是否为目录
isFile                是否为文件
isHidden                是否为隐藏文件或目录
lastModified            获取文件最后修改时间
length                获取文件长度
listFiles            获取目录的子目录、文件
listRoots            获取所有磁盘分区
mkdir                创建目录
mkdirs                创建多级目录
    <3> e.g
    // 获取当前文件系统的所有磁盘分区
 File[] disks = File.listRoots();
 System.out.println("系统磁盘分区情况：");
 for (int i = 0; i < disks.length; i++) {
     File disk = disks[i];
     System.out.println(disk.getAbsolutePath() + "盘\t总空间:"
             + disk.getTotalSpace() / 1024 / 1024 / 1024 + "G\t剩余空间"
             + disk.getFreeSpace() / 1024 / 1024 / 1024 + "G");
 }
  2. 流
    <1> 分类
    字节流和字符流
    <2> 文本文件和二进制文件
    二进制文件，指文件无字符编码格式，均由字节（Byte）组成，图片文件、word文档等均为二进制文件
文本文件是一种特殊的二进制文件，也由字节组成，但需要通过特定的字符编码格式读取或写入，否则会出现乱码
后缀名为txt的文件就是典型的文本文件
    <3> 字节流和字符流
    字节流读取二进制文件
字符流读取文本文件
  3. 字节流
    <1> 字节输入流(FileInputStream) 
    步骤:
     打开文件->按字节读取文件->关闭文件
注:
     a. 打开文件通过向FileInputStream的构造函数传入文件名或File对象完成
        如果文件不存在会抛出FileNotFoundException异常    
     b. 按字节读取文件内容通过FileInputStream实例的read方法实现
        read方法按顺序读取文件中的字节，返回int型，返回值在-1~255之间
        如果返回-1说明已读到文件尾，当读取发生错误时read方法会抛出IOException异常
     c. 关闭文件通过FileInputStream实例的close方法实现
        关闭时遇到错误会抛出IOException异常
    <2> 字节缓冲输入流
    e.g
     File file = new File("C:/1.txt");
     try {
         FileInputStream fis = new FileInputStream(file);
         BufferedInputStream bis = new BufferedInputStream(fis);
         long startTime = System.currentTimeMillis();
         int i;
         while ((i=bis.read())!=-1) {
         //为保证测试的准确，去掉打印输出
         }
         long endTime = System.currentTimeMillis();
         System.out.println("总用时：" + (endTime - startTime));
         bis.close();
         } catch (FileNotFoundException e) {
             System.out.println("文件未找到！");
         } catch (IOException e) {
             System.out.println("文件操作时发生异常！");
         }
注:
     ①缓存输入流不能独立使用，需要通过文件输入流进行构建
     ②应使用缓存输入流实例读取文件，如继续使用fis（文件输入流实例），则绕过了缓存，性能不会得到提升
     ③关闭文件时只需要关闭缓存输入流实例即可
    <3> 字节输出流(FileOutputStream)
    e.g
     File file = new File("C:/1.txt");
     try {
         FileOutputStream fos = new FileOutputStream(file);
         //65,66,67分别为A,B,C的ASCII码
         fos.write(65);
         fos.write(66);
         fos.write(67);
         fos.close();
     } catch (IOException e) {
     System.out.println("文件操作时发生异常！");
     }
注:
     在使用文件输出流时，如果目标文件不存在，会创建该文件;如果目标文件存在会覆盖该文件的内容
     如果当目标文件存在时不想覆盖原内容，而是进行追加输出
     可以按如下方式构架FileOutputStream实例：
     FileOutputStream fos = new FileOutputStream(file, true);
    <4> 缓存输出流(BufferedOutputStream)
    e.g
     File file = new File("C:/1.txt");
     try {
         FileOutputStream fos = new FileOutputStream(file);
         BufferedOutputStream bos = new BufferedOutputStream(fos);
         bos.write(65);
         bos.write(66);
         bos.write(67);
         //flush方法用于强制将缓存中内容输出
         bos.flush();
         bos.close();
     } catch (IOException e) {
       System.out.println("文件操作时发生异常！");
     }
  4. 字符流
    <1> 字符流会对文件内容编码，所以不能用于读取二进制文件
