Java学习——JSP简介

发表于 2019-05-26 | 更新于 2019-07-29 | 分类于 Java | 评论数：
  1. WEB容器
    <1> 简介
    Web(World Wide Web)容器(Container)是一种服务程序
在服务器一个端口就有一个提供相应服务的程序
而这个程序就是处理从客户端发出的请求，并向客户端作出响应
    <2> 涵盖的框架技术
    JSP、Servlet
    <3> 容器
    Apache公司的Tomcat容器
IBM公司的WebSphere容器
Oracle公司的Weblogic容器
  2. C/S和B/S结构
    <1> C/S结构
    C/S（Client/Server）结构，即客户机和服务器结构
它会将任务合理分配到Client端和Server端来实现
但它是胖客户端架构，作为客户端的部分需要承受很大的压力
C/S需要安装客户端，比如QQ
    <2> B/S结构
    B/S（Browser/Server）结构即浏览器和服务器结构
用户工作界面是通过www浏览器来实现
B/S结构可以看作是瘦客户端，只是把显示的较少的逻辑交给了Web浏览器
事务逻辑数据处理在放在了Server端
B/S结构的系统无须特别安装，只要Web浏览器即可，比如浏览的网站
    <3> 区别
    B/S无需安装，客户端不需要安装有浏览器即可
B/S具有分布性特点，可以随时随地进行查询、浏览等业务处理
B/S升级维护便捷，无需升级多个客户端，升级服务器即可
C/S客户端需要安装专用的客户端软件
C/S维护升级成本高，进行一次维护升级，需要所有客户端的程序进行重新安装
    <4> 注
    JSP采用的是B/S结构
  3. HTTP协议
    <1> 概念
    Http是超文本传输协议（Http，HyperText Transfer Protocol)
是一个基于请求与响应模式的、无状态的、应用层的协议
它的作用是从www服务器传输超文本到本地浏览器，浏览器再解析超文本
将解析后的超文本呈现成网页，供用户浏览
    <2> 过程
    由Http客户端发起一个请求，请求到一个服务器指定的端口（默认是80端口）的TCP连接
Http服务器则在那个端口监听客户端发送过来的请求
一旦收到请求，服务器向客户端发回一个状态行，比如"HTTP/1.1 200 OK"
和响应的消息，消息的消息体可能是请求的文件、错误消息、或者其它一些信息
    <3> URL
    URL：(Uniform Resource Locator 的缩写，统一资源定位符
http://www.it.cn:80/course/index.jsp?id=1
协议://服务器名称:端口号/路径/文件名?查询
  4. JSP简介
    <1> 简介
    JSP就是将java代码写在HTML页面中
而Java代码需要写在以“<%”开头，以“%>”结尾的区域内
    <2> 指令
    JSP指令用来设置整个JSP页面相关的属性
在JSP中指令分为page、include和taglib三种指令
  5. page指令
    <1> 用途
    设置JSP页面的全局属性
    <2> 格式
    <%@ page 属性= "值" 属性="值" ….%>
    <3> 常用属性和值
    language=”java”    告诉Web容器用什么语言来编译JSP页面，默认值为java
import=”java.util.*”    导入JSP页面中需要使用的类
pageEncoding=”UTF-8”    表示JSP页面的编码方式
    <4> 注意
    在page指令中除了import属性之外，其他的属性在一个JSP页面中只能出现一次
<%@ page language="java" import="java.util.*,java.sql.* " pageEncoding="UTF-8" %>
  6. Scriptlet（小脚本）
    <1> 简介
    scriptlet只是普通的Java代码
    <2> 内置对象
    内置对象是指由Web容器实例化的，在JSP页面中程序员无需实例化就可以直接使用的对象
    <3> 九个内置对象
    request、response、out、session、application、pageContext、page、exception、config
    <4> e.g
    <%
     String name = "fafa";
     out.println(name); 
%>
out内置对象用于向页面上输出内容
  7. 表达式块
    <1> 用途
    向页面上输出内容
    <2> 格式
    <%=输出内容%>
    <3> e.g
    <% String name = "fafa"; %>
<%= name %>
  8. 声明
    <1> 用途
    声明是在JSP页面中声明方法或类的成员变量
    <2> 格式
    <%! 声明内容 %>
    <3> e.g
    <%!
       public String sayHello(){
       return "Hello,JSP";
       } 
%>
可以在表达式中调用并输出内容：
<%=sayHi()%>
  9. 注释
    <1> 格式
    <%-- 注释内容 --%>
    <2> 注
    JSP注释在客户端查看源代码不可见，HTML可见
