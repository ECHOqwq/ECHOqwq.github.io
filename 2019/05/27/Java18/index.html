Java学习——内置对象request

发表于 2019-05-27 | 更新于 2019-06-07 | 分类于 Java | 评论数：
  1. 总述
    <1> JAVAEE开发模式:
    基于请求与响应的运营模式
    <2> 请求和响应 
    请求是指客户端浏览器向服务器发出请求
响应是指服务器反馈给客户端的页面
    <3> 发送请求 
    在客户端浏览器发出请求时，经常需要向服务器端发送数据
被发送的数据通常是以表单提交的方式提交到服务器的
    <4> 研究对象
    研究客户端浏览器如何向服务器发送数据
  2. HTML表单
    <1> 定义
    表单是实现客户端输入数据的HTML标签
    <2> e.g
    <form id="form1" name="form1" method="post" action="controller.do">
</form>
    <3> 注
    form是表单标签
id是表单在浏览器端的唯一表示，通常由CSS和JavaScript使用id
name是表单的名称
method属性表示客户端浏览器向服务器的请求方式，其值可以是get和post
action属性表示表单的数据提交到哪里，通常是接收表单数据的URL。
    <4> method属性中post和get的区别
    提交方式不同
传输数据的大小不同
安全性：post安全性比get高
例：
通过GET提交数据，用户名和密码将明文出现在URL上
因为登录页面有可能被浏览器缓存，其他人查看浏览器的历史记录
那么别人就可以拿到你的账号和密码了
  3. HTTP请求报文
    <1> 定义
    浏览器向服务器发出请求时，浏览器向服务器提交的数据以及一些附加的信息
HTTP请求报文由三个部分组成
分别是：状态行、消息报头、请求正文
    <2> 例
    <%@ page language="java" contentType="text/html; charset=utf-8" pageEncoding="utf-8"%>
<!doctype html><html><head><meta charset="utf-8"></head>
<body>
<form id="form1" name="form1" method="post" action="doadd.jsp?id=1">
<p>姓名：<input type="text" name="trueName" id="trueName" /></p>
<p>年龄：<input type="text" name="age" id="age" /></p>
<p><input type="submit" name="submit" id=" submit " value="提交"/></p>
</form>
</body>
</html>
运行后可在浏览器页面使用开发人员工具(F12)查看HTTP报文
    <3> 包含
    a.General
    General中包含请求地址、请求方式、服务器响应状态、服务器地址
b.Request Headers
    Request Headers称为请求头，在请求头附加了一些其它信息，如Cookie、Content-Type、User-Agent、Accept-Language等
c.Query String Parameters
    Query String Parameters称为查询字符串参数，查询字符串参数是指浏览器地址栏中的URL中?后面传递的参数
    上例的action="doadd.jsp?id=1"中的id=1就是查询字符串参数
    Get方式提交的数据以查询字符串参数方式提交到服务器
d.Form Data
     Form Data是指表单中提交的数据
     上例中Form Data数据包含trueName=admin, age=21
     post提交的数据以Form Data方式提交到服务器
    <4> 请求过程
    a.建立TCP连接 (Web浏览器首先要通过网络与Web服务器建立连接，该连接是通过TCP来完成的，该协议与IP协议共同构建Internet，即著名的TCP/IP协议族)
b.Web浏览器向Web服务器发送请求命令(前提：TCP连接需建立)
c.Web浏览器发送请求头信息
d.Web服务器应答
e.Web服务器发送响应头信息
f.Web服务器向浏览器发送数据
g.Web服务器关闭TCP连接
  4. 内置对象request
    <1> 定义
    它是javax.servlet.http.HttpServletRequest类的对象，用于让服务器获取客户端提交的数据
    <2> 方法
    HttpSession getSession()    获得Session对象
Cookie getCookies()    获得cookies
Object getAttribute(String key)    根据key获取在request存储范围中的值
void setAttribute(String key,Object obj)     向request范围中存储键值对
String getParameter(String name)    根据name获取表单域或URL的参数值
String[] getParameterValues(String name)    根据name获取同名表单域的值
void setCharacterEncoding(String encoding)    设置request的字符编码格式
RequestDispatcher getRequestDispatcher()    获取请求转发对象
String getProtocol()    返回请求用的协议类型及版本号
String getScheme()    返回请求用的计划名,如:http及ftp等
String getServerName()    返回接受请求的服务器主机名
int getServerPort()    返回服务器接受此请求所用的端口号
String getRemoteAddr()    返回发送此请求的客户端IP地址
String  getRemoteHost()    返回发送此请求的客户端主机名
StringgetRealPath(String path)    返回一虚拟路径的真实路径
String request.getContextPath()    返回上下文路径
    <3> 注
    因为它是内置对象，所以使用时不用实例化
  5. 内置对象out
    <1> 定义
    用于向客户端浏览器输出数据，它是javax.servlet.jsp.JspWriter类的对象
    <2> 方法
    print(Object)    向客户端浏览器输出数据
void flush()    清空流
void close()    关闭输出流
