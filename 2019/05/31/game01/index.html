利用easyx进行游戏开发基础

发表于 2019-05-31 | 分类于 Game | 评论数：

1.头文件的使用
graphics.h
2.绘制图形窗口
(1)initgraph();
(2)closegraph();
3.图形窗口额外的东西
(1)坐标系统
横着是x轴，由左至右增大；竖着是y轴，从上到下增大
(2)窗口的颜色设置
a.背景颜色
setbkcolor(颜色)
b.颜色的知识点
颜色宏:大写英文颜色单词 BLACK BLUE YELLOW GREEN
RGB 配置颜色 三原色配置 RGB(196,15,196)
注:可去画图中查看所需颜色对应的三原色配置
c.第一次设置背景颜色需要刷新(cleardevice())
(3)窗口文字输出
a.字符串
outtextxy(int x,int y,”字符串”); //多字节
b.数字类输出
用sprintf函数把数字转化成字符串
e.g
int i = 1, j = 2;
char a[20] = “”;
sprintf(a,”%d,%d”,i,j);
outtextxy(0,0,a);
(4)窗口绘制函数
a.画线 line(int x,int y,int xx,int yy); //两点的坐标
b.画非填充矩形 rectangle (int x,int y,int xx,int yy);//左上角右下角坐标
c.画填充矩形 fillrectangle(int x,int y,int xx,int yy);
d.填充颜色设置 setfillcolor(颜色)
4.鼠标操作
(1) 知道鼠标操作反馈给计算机的是一个值
(2) 鼠标变量存储鼠标反馈的值
MOUSEMSG 类型的变量去存储
(3) 获取反馈 GetMouseMsg();
(4) 分类处理 针对 uMsg 分类处理
综合框架
MOUSEMSG m; //声明一个鼠标变量
while(1)
{
m=GetMouseMsg(); //获取鼠标消息
switch (m.uMsg)
{
case WM_LBUTTONDOWN:
……;
break;
case WM_RBUTTONDOWN;
break;
case WM_MOUSEMOVE:
break;
}
}
5.贴图函数
(1)图片存储到IMAGE类型变量
IMAGE img;
(2)图片用变量存储
loadimage(&img,”路径”);
(3)显示图片
putimage(int x,int y,&img);
(4)图片问题的处理
a.图片太大，窗口容不下:窗口配合图片的尺寸
b.缩放图片大小适应窗口: loadimage(&img,”x.jpg”,窗口宽，窗口高);
c.initgraph(img.getwidth();img.getheight());
d.多个图片的时候去掉背景的问题
背景图:黑底食物
掩码图:白底黑洞
SRCAND方式 贴掩码图
SRCPAINT方式 贴背景
先贴掩码图再贴背景图
