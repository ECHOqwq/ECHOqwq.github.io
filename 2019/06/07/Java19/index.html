Java学习——内置对象request

发表于 2019-06-07 | 分类于 Java | 评论数：

1.响应报文
<1> 定义
HTTP响应报文是指服务器向浏览器响应的数据以及一些附加的信息
<2> 组成
状态行、消息报头、响应正文

  1. 内置对象response
    <1> 定义
    response是JSP的内置对象，它是javax.servlet.http.HttpServletResponse类的实例
用于动态响应客户端的请求，控制发送给用户的信息
    <2> 方法
    PrintWriter getWriter()    返回可以向客户端输出字符的一个对象
void setContentType(String type)    设置响应的MIME类型
void sendRedirect(java.lang.String location)    重新定向客户端的请求
  2. 页面跳转-重定向
    <1> 页面跳转
    页面跳转是指从一个页面转到另外一个页面
    <2> 重定向
    在JSP中页面跳转可以通过response内置对象的sendRedirect()方法实现页面跳转
这种页面跳转的方式称为重定向
    <3> 重定向的原理
    在jsp页面中执行response.sendRedirect(path)时，服务器向客户端响应状态码302和path
302状态码表示重定向，path表示重定向的URL
当浏览器接收到状态码302后，就向path表示的URL发出新的请求
重定向是浏览器客户端重新向服务器发出的请求
重定向后浏览器地址栏的URL显示为重定向后的URL
  3. 内置对象exception
    <1> 定义
    它是java.lang.Throwable类的对象
该内置对象代表了JSP页面在运行时的异常，该对象只有在错误页面中才可以使用
    <2> 错误页面
    错误页面指的是在JSP的page指令中指定属性isErrorPage的值为true的页面
    <3> 全局异常配置
    可以在web.xml中配置全局异常处理，不必在每个jsp页面中配置errorPage属性
