Java学习——EL和JSTL

发表于 2019-06-07 | 更新于 2019-07-29 | 分类于 Java | 评论数：
  1. 总述
    <1> 用途
    输出通常使用EL和JSTL实现，使用EL和JSTL实现输出是为了在界面上不出现Java代码，而使得界面上只有标签

  2. EL表达式
    <1> 简介

    EL表达式又称为表达式语言，在JSP页面中使用EL表达式，可以简化对变量和对象的访问
EL表达式都是以“${”开始，以“}”结束，EL表达式会将表达式中的结果在页面上输出

    <2> EL表达式支持运算符

    使用EL表达式进行算术运算，在EL表达式中支持+、-、*、/、%运算
支持关系运算符操作，还可以使用字符来表示关系运算符，支持逻辑运算
使用empty运算符检测一个值是否为null或者为空
还支持三元表达式方式的条件选择运算

    <3> e.g

    <body>
      12 + 15 = ${12+15}<br>
      12 * 15 = ${12*15}<br>
      12 - 15 = ${12-15}<br>
      12 / 15 = ${12/15}<br>
      12 % 15 = ${12%15}<br>
</body>

<body>
      12==15  ${12==15}<br>
      12<15  ${12<15}<br>
      12>15  ${12>15}<br>
      12<=15  ${12<=15}<br>
      12>=15  ${12>=15}<br>
      12!=15  ${12!=15}
</body>

<body>
      12==15  ${12 eq 15}<br>
      12<15  ${12 lt 15}<br>
      12>15  ${12 gt 15}<br>
      12<=15  ${12 le 15}<br>
      12>=15  ${12 ge 15}<br>
      12!=15  ${12 ne 15}
</body>

<body>
      ${12 < 15 && 12 < 15 }<br>
      ${12 < 15 and 12 < 15 }<br>
      ${12 < 15 || 12 > 15 }<br>
      ${12 < 15 or 12 >15 }<br>
      ${!(12 < 15) }<br>
      ${not (12 < 15) }
</body>

<body>
  <%
    List list = new ArrayList();
    list.add("tom");
    pageContext.setAttribute("str", null);
    pageContext.setAttribute("list", list);
  %>
  ${empty str }
  ${empty list}
</body>

<body>
      ${12 < 15 ? 'yes':'no' }
</body>

    <4> EL访问对象(按照顺序寻找)

    EL表达式默认会从pageContext、reqest、session、application中
按照顺序寻找名称匹配的对象，如果找不到，则不显示任何内容

    <5> e.g 

    <body>
  <%
    pageContext.setAttribute("name","tom");
    request.setAttribute("age","22");
    session.setAttribute("address","中国");
    application.setAttribute("score","75.5");
  %>
  ${name}<br>
  ${age}<br>
  ${address}<br>
  ${score}<br>
</body>

    <6> EL访问对象(从指定空间中访问)
    <7> e.g

    <body>
   <%
      pageContext.setAttribute("name","tom");
      request.setAttribute("age","22");
      session.setAttribute("address","中国");
      application.setAttribute("score","75.5");
    %>
    ${pageScope.name}<br>
    ${requestScope.age}<br>
    ${sessionScope.address}<br>
    ${applicationScope.score}<br>
</body>

    <8> EL访问自定义类

    如果直接输出自定义类的对象，EL表达式将会访问类的toString方法并将返回值进行输出
还可以很方便的使用EL表达式获得对象的属性值
通过点“.”运算符来获取对象user中的name属性的值
再次需要注意要访问的属性名不一定是和对象的属性名相同
而是要和get方法的方法名去掉“get”后的名字相同，把开头改成小写
  3. JSTL核心标签库
    <1> 总
    JSP标准标签库，是由SUN公司提供的简化JSP页面设计的标签
JSTL是由Core、I18N、SQL、XML、Functions五个核心标签库组成
其中最重要的是Core标签库和I18N标签库中的格式化标签库
    <2> Core标签库
    Core标签库中包含了基本标签、条件标签、迭代标签
在使用Core标签时需要在JSP页面中使用taglib指令导入Core标签库：
    <%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
Core标签库中的基本标签包括<c:out>、<c:set>、和<c:remove>标签
    <3> c:set标签
    用于在某空间中（pageContext、request、session、application）声明变量并赋值
var    设置保存value的变量名
value    变量的值或将表达式的值作为变量值
scope    变量存储的空间，值为page、requqest、session、application，默认为page
target    要设置属性的对象，必须为JavaBean（属性有set方法）或java.util.Map对象
property    要设置target对象属性的名称
    <4> c:out标签
    用于将空间内的变量进行输出
value    输出值的变量名
escapeXml    确定在结果字符串中的字符“< > ' " &”等符号应该被转换为对应的字符引用或预定义实体引用，默认值为true
default    如果value为null，则显示default中的值
    <5> c:remove标签
    用于将变量从空间内移除
var    要移除的变量名称
scope    var的空间名称，默认为page
    <6> c:if标签
    和Java语言中的if语句功能是相同的
test    条件语句，用于判断标签体是否可以被执行
var    将test条件语句执行的结果保存在var声明的变量中
scope    var的存储空间，默认为page
    <7> c:choose、c:when和c:otherwise标签
    实现Java语言中的if/else功能
<c:choose>标签是<c:when>和<c:otherwise>标签的父标签，在<c:choose>标签中只能出现这两个子标签
    <8> c:forEach标签
    迭代标签
var    存储当前迭代元素的变量名
items    被迭代的集合或数组
varStatus    迭代状态对象的变量名
begin    指定迭代开始的索引
end    指定迭代结束的索引
step    迭代的步长
  4. JSTL格式化标签库
    <1> 总
    需要在使用的JSP页面中使用taglib指令导入格式化标签：
<%@ taglib prefix="fmt" uri="http://java.sun.com/jsp/jstl/fmt" %>
    <2> 两大标签
    在JSTL格式化标签中最重要的是<fmt:formatNumber>标签和<fmt:formatDate>标签
分别代表对数字和日期的格式化输出
    <3> fmt:formatNumber
    value    要格式化的值
type    要按照什么类型（数字、货币、百分比）去格式化。默认值为number。货币为currency，百分比为percent
pattern    自定义的格式化样式
currencyCode    ISO 4217货币代码。只适用与格式化货币，其他格式化类型则忽略此属性。
currencySymbol    货币符号，比如“￥”。只适用于格式化货币，其它格式化类型则忽略此属性。
groupingUsed    格式化的输出是否采用分组方式输出。默认为true
maxIntegerDigits    指定格式化输出的整数部分的最大数字位数
minIntegerDigits    指定格式化输出的整数部分的最小数字位数
maxFractionDigits    指定格式化输出的小数部分的最大数字位数
minFractionDigits    指定格式化输出的小数部分的最小数字位数
var    将格式化后的结果存储到空间内的变量名
scope    将var变量存储到的空间名称,默认为page
    <4> fmt:formatDate
    value    要格式化的日期或时间，类型为java.util.Date
type    指定value中的日期部分或时间部分要进行格式化还是全部都要格式化。值为time|date|both，默认为date
ateStyle    日期的预定义格式化样式，有default | short | medium | long | full五种样式，默认为default
timeStyle    时间的预定义格式化样式，有default | short | medium | long | full五种样式，默认为default
pattern    自定义格式化时间或日期样式
timeZone    使用时区
var    将格式化后的值保存到空间内的变量值
scope    var变量保存的空间
