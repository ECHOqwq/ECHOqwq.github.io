Mysql——数据库、数据表

发表于 2019-07-26 | 更新于 2019-07-27 | 分类于 mysql | 评论数：
  1. 简介窗口界面
    <1> 数据库导航窗口
    可以查看当前数据库管理系统中所有的数据库
当前正在使用的数据库等信息
    <2> 询问窗口
    SQL语句的编辑窗口，在询问窗口中可以定义各种SQL语句
每一条独立的SQL语句都以英文分号结束
完整的SQL语句可以在点击执行按钮进行执行
    <3> 执行结果窗口
    在这个窗口中，主要显示SQL语句执行的信息
包括执行条数、执行的时间、查询的数据等等信息
  2. 常用快捷键
    <1> Ctrl+M 创建一个新的连接
    Ctrl+N   使用当前设置新建连接
Ctrl+F4   断开当前连接
Ctrl+T   新建查询编辑器
Ctrl+Y   重做
Ctrl+Z   撤销  
Ctrl+O   在 SQL 窗口打开一个 SQL 文档
F8   执行当前查询并编辑结果集
F9   执行当前查询(默认)
Ctrl+F9   执行选定查询(默认)
Shift+F9     执行全部查询(默认)
Ctrl+D   创建数据库
F6   更改数据库/数据表的结构/索引/视图/存储过程/函数/触发器/事件
F2   重命名数据表/视图/触发器/事件
Ctrl+U   添加用户
Ctrl+Alt+U   编辑用户
Ctrl+Shift+W  管理用户权限
  3. MySQL服务启动与关闭
    <1> 启动
    win+R->启动cmd->输入net start mysql回车
    <2> 停止
    同上启动cmd->输入net stop mysql
  4. SQL (结构化查询语言)
    <1> 用途
    用于数据存取、数据更新及数据库管理等操作
以此对数据库的表进行增删改查的操作
    <2> 流程
    在MySQL开发中
客户端把SQL语句发送给服务器
服务器对SQL语句进行编译、执行
把执行的结果返回给客户端
    <3> 组成
    1. 数据定义语言（DDL）
   包括CREATE（创建）命令、ALTER（修改）命令、DROP（删除）命令等
2. 数据操纵语言（DML）
   包括INSERT（插入）命令、UPDATE（更新）命令、DELETE（删除）命令、SELECT … FOR UPDATE（查询）
3. 数据查询语言（DQL）
   包括基本查询语句、Order By子句、Group By子句等
4. 事务控制语言（TCL）
   包括COMMIT（提交）命令、SAVEPOINT（保存点）命令、ROLLBACK（回滚）命令。
5. 数据控制语言（DCL）
   GRANT（授权）命令、REVOKE（撤销）命令
  5. 创建数据库[DATABASE]
    <1> 注释
    以#开头
    <2> 创建数据库
    e.g
  CREATE DATABASE company_info;
注:执行快捷键Ctrl+F9
    <3> 使用数据库
    USE db_name;
e.g
  USE company_info;
    <4> 删除数据库
    DROP DATABASE db_name;
e.g
  DROP DATABASE company_info;

  6. 创建数据表[TABLE]
    <1> MySQL常见数据类型

    SMALLINT    小的整数
MEDIUMINT    中等大小的整数
INT/ INTEGER    普通大小的整数
BIGINT    大整数
FLOAT    小(单精度)浮点数
DOUBLE    普通大小(双精度)浮点数
DATE    日期
DATETIME    日期和时间的组合
TIMESTAMP    时间戳
TIME    时间
YEAR    两位或四位格式的年
CHAR(M)    固定长度字符串，当保存时在右侧填充空格以达到指定的长度
VARCHAR(M)    变长字符串

    <2> 用途

    存储各种各样的数据于一个表中

    <3> 创建数据表

    CREATE TABLE tab_name (
     col_name(字段名称) datetype(数据类型),
     ……
);
e.g
 #创建部门表
 CREATE TABLEdept(
 Deptno INT,
 Dname VARCHAR(14),
 Loc VARCHAR(13)
 );

 # 创建员工表
 CREATE TABLE emp(
 empNo INT,
 ename VARCHAR(10),
 job VARCHAR(10),
 mgr INT,
 hirdate DATETIME,
 sal DOUBLE,
 comm DOUBLE,
 deptno INT
 );

    <4> 删除数据表

    DROP TABLE tab_name;
e.g
   DROP TABLE emp;
  7. 数据表的基本操作
    <1> 增加数据
    1. # 使用INSERT语句给表中所有字段添加值
     INSERT INTO tab_name
     VALUES(col_value1, col_value2, ...);
2. # 使用INSERT语句，给表中指定的列添加值
     INSERT INTO tab_name(col1, col2, col3, ...)
     VALUES(col1_value, col2_value, col3_value,...);
3. # 给dept表中添加数据
     INSERT INTO dept 
     VALUES(10, '人事部', '北京');
     INSERT INTO dept(deptno,dname, loc) 
     VALUES(20,'软件部', '深圳');
4. # 给emp表中添加数据
     INSERT INTO emp(empNo, ename, job, mgr, hirdate, sal, comm, deptno) 
     VALUES(1001, '宋江', '董事长', null ,'2001-1-1', 6000, 10000, 10);
     INSERT INTO emp(empNo, ename, job, mgr, hirdate, sal, comm, deptno) 
     VALUES(10011, '卢俊义', '董事助理', 1001,'2003-1-1', 3000, 1000, 10);
     INSERT INTO emp(empNo, ename, job, mgr, hirdate, sal, comm, deptno) 
     VALUES(10012, '吴用', '董事助理', 1001,'2001-1-1', 4000, 4000, 10);
     INSERT INTOemp(empNo, ename, job, mgr, hirdate, sal, comm, deptno) 
     VALUES(2001, '林冲', '项目经理', 1001,'2003-3-1', 6000, 5000, 20);
    <2> 修改数据
    # 修改表tab_name中符合条件condition的字段col_name的值为col_value
UPDATE tab_name SET col_name=col_value,...WHERE condition;
e.g 
   #修改emp表中员工李逵的奖金，从1000升为3000
    UPDATE emp SET comm = 3000 WHERE ename='李逵';
    <3> 删除数据
    #删除数据表中符合条件的数据
DELETE FROM tab_name WHERE condition;
e.g 
   # 删除员工表中李逵的信息
    DELETE FROM emp WHERE ename='李逵';
    <4> 查找数据
    #查看tab_name表中所有列
SELECT * FROM tab_name;
e.g  
   #查看员工表emp中所有信息
    SELECT * FROM emp;
   #查看部门表dept中所有信息
    SELECT * FROM dept;
  8. 使用cmd进行数据库的备份与恢复
    <1> 备份
    #数据库备份语法结构
    Mysqldump –hxxx.xxx.xxx.xxx –uusername –ppassword db_name > dir/file.sql
注:
   在命令行窗口，执行使用mysqldump命令
   -h指定数据库所在服务器的ip地址
   -u指定登录数据库的用户名
   -p指定登录数据库的密码，并指定要备份的数据库名称db_name
   使用输出目标操作符>指定输出的文件具体路径dir/file.sql
    <2> 还原
    #数据库还原语法结构
    mysql–hxxx.xxx.xxx.xxx –uusername –ppassword db_name < back.sql
注:
    使用mysql命令，将指定的备份文件back.sql还原到ip为xxx.xxx.xxx.xxx的数据库中
    数据库登录账号username，数据库登录密码password
  9. 使用SQLYog进行数据库的备份与恢复
    <1> 备份
    1. 右键点击要备份的数据库，在弹出的菜单中，选择【备份/导出】-->【备份数据库、转储到SQL...】
2. 弹出的对话框中，选择备份【结构和数据】，勾选要备份的对象
   然后选择备份到的文件，如M:\back.sql；点击确定即可
    <2> 还原
    使用SQLYog登陆数据库 
在SQLYog中以还原的数据库名称来创建新的数据库，然后右键点击该数据库，选择【导入】-->【执行SQL脚本】
在弹出的导入对话框中，选择要还原的数据文件所在的位置，点击执行
还原完成后，检查数据库数据还原是否正常即可
