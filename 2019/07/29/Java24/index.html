JSP基础知识整理2

发表于 2019-07-29 | 更新于 2019-07-31 | 分类于 Java | 评论数：
  1. EL表达式
    <1> 简介
    EL，Expression Language，表达式语言
是一种在JSP页面中获取数据的简单方式
JSP页面的任何静态部分可以通过${expression}获取到指定表达式的值

  2. 获取数据

    <1> 从四大域中依次查找数据
        EL只能从pageContent、request、session、application四大域属性空间中获取数据
<2> e.g
    <%
        String username = "hahaha";
        pageContext.setAttribute("username",username); //重要语句
        request.setAttribute("username",username);//重要语句
        session.setAttribute("username",username); //重要语句
        application.setAttribute("username",username); //重要语句
        //username是内容
    %>
    username = ${username}<br>
    注:
       如果去掉重要语句，out: username =
       如果保留重要语句，out: username = hahaha
<3> 顺序
        依次按照由小到大的范围从四大域中查找指定名称的属性值
<4> 从指定域中获取数据
        利用EL的四个域属性空间相关的内置对象
        pageScope 从page范围域属性空间中查找指定的key
        requestScope 从request范围域属性空间中查找指定的key
        sessionScope 从session范围域属性空间中查找指定的key
        applicationScope 从application范围域属性空间中查找指定的key
<5> EL访问bean的属性
        若有一个包名为java.beans，要访问其中一个名为Student的类
        先import = "java.beans.Student"
        <%
                Student student = new Student("张三",20);
                pageContext.setAttribute("student",student);
        %>
        student = ${student } <br>
        name = ${student.name } <br>
        age = ${student.age } <br>
        若访问为null的对象的属性，EL不会抛出空指针异常的信息，仅仅不显示而已\
        即若Student类里不存在grade的属性，则
        grade = ${student.grade } <br>    out:grade = 
<6> EL访问数组中的元素
        EL可以通过索引的方式获取到指定索引的元素
        若数组不存在指定索引的元素，不会抛出数组越界异常
        <%
            String[] names = {"张三","李四"}
            pageContext.setAttribute("names",names);
        %>
        names[1] = ${names[1] } <br>
        names[5] = ${names[5] } <br> 
        <!--结合上面的访问bean属性访问数组中的元素-->
        <%
            Student[] students = new Student[2];
            students[0] = new Student("张三",20);
            students[1] = new Student("李四",21);
            pageContext.setAttribute("student",student);
        %>
        student[1].name = ${students[1].name} <br>
<7> EL访问list中的元素
        通过索引的方式可以获取list中指定索引的元素
        若list中不存在指定索引元素，系统不会抛出越界异常
        但无法访问set，因为set中没有索引的概念
        <%
                Student student1 = new Student("张三",20);
                Student student2 = new Student("李四",24);

                List<Student> students = new ArrayList();
                students.add(student1);
                students.add(student2);
                students.add(student3);
                pageContext.getAttribute("stus",students);
        %>
        students[2].name = ${stus[2].name} <br> 

        <%
                List<String> names = new ArrayList<>(); // <>可写可不写
                names.add("张三");
                names.add("李四");
                pageContext.setAttribute("names",names);
        %>
        names[2] = ${names[2]} <br>
<8> EL访问Map
        EL通过${attributeName.mapKey}的方式可以获取指定的key的值
        <%
                Map<String, Object> map = new HashMap();
                map.put("school",new School("清华大学","北京海定"));
                map.put("mobile","1234567");
                map.put("age",20);
                pageContext.setAttribute("map",map);
        %>
        school.name = ${map.school } <br>
        mobile = ${map.mobile } <br>
        age = ${map.age } <br>
  3. 运算符
    <1> 算术运算符
        支持+ - × ÷ 但不支持++ --
<2> 关系运算符
        支持== != > >= < <=
<3> 逻辑运算符
        支持! && || not and or
<4> 取值运算符
        支持[] 点号
<5> empty变量
        <%
                Map<String, Object> map = new HashMap();
                map.put("school",new School("清华大学","北京海定"));
                map.put("mobile","1234567");
                map.put("age",20);
                pageContext.setAttribute("map",map);
        %>
        empty name = ${empty name } <br>
        <!--empty对没有被定义的变量的引用返回true-->
        <!--empty对于值为null的变量的引用也返回true-->
        <!--empty对于空串的String的引用的运算结果也是true-->
        <!--empty对于没有元素的数组或集合的运算结果也是true-->
  4. EL的内置对象
    <1> pageContext
        该pageContext和JSP的pageContext是同一个对象
        通过该对象可以获取request、response、session、servletContext、servletConfig等对象
        这些对象在EL中只能通过pageContext获取
        其余之后再补(19p)
  5. 自定义EL函数
    <1> 创建过程
        先创建一个普通.java文件，并在里面创建一个函数写出需要实现的功能
        该类及其函数，需要在一个扩展名为.tld的XML文件中进行注册
        tld: tag library definition，标签库定义
        XML文件是需要配置文件头部来进行约束的
        可以在Tomcat的安装目录下:tomcat-9\webapps\examples\WEB-INF\jsp2
        文件为 jsp2-example-taglib-tld文件中进行复制
        这个.tld的XML文件需要定义在当前web项目的WEB-INF目录下
        其余再补(21p)
