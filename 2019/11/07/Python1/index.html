Python基础

发表于 2019-11-07 | 分类于 Python | 评论数：
  1. 类型转换函数
    <1> chr(x)
    把一个整数转换成一个字符，整数为字符的ASCII码值
    <2> ord(x)
    把一个字符转换成它的ASCII码值
    <3> hex(x)
    把一个整数转换成十六进制字符串
    <4> oct(x)
    把一个整数转换成八进制字符串
    <5> eval(str)
    把字符串str当做有效表达式求值，并返回计算结果
e.g
    x = 5
    y = 3
    eval('x-y')
    out: 2
  2. 并行赋值
    e.g
    x,x = 3,x*20
print(x)
out: 60
x = 3
x = x*20
所以x为60
    应用：
    常用于交换两个变量的值
x,y = y,x
  3. 循环语句+else
    <1> while…else
    else语句会在循环正常执行完后执行(不管是否执行循环体)
    <2> for…else
    如果执行for语句没有遇到break，则会在循环结束后执行else
  4. 列表
    <1> 列表切片
    列表名[开始索引:结束索引:步长]
即索引到结束索引前一元素，步长没定义默认为1
    <2> 添加元素
    a. "+"
    将一个新列表添加在原列表的尾部
b. extend()
    将一个新列表添加在原列表的尾部
c. 二者区别
    "+"是创建了一个新列表，把原列表和增加列表都放进去
    extend()是向列表尾部添加一个新的元素，速度更快
d. append()
    在列表尾部添加一个新的元素
e. insert()
    把一个元素插入列表的指定位置，第一个参数是插入位置，第二个参数是插入元素
    <3> 检索元素
    index(value,[,start,[,end]])
其中start和end是开始位置和结束位置，start默认为0，end默认为列表长度
    <4> count()
    统计特定元素在列表中的出现次数
    <5> in
    看该元素是否在列表中
    <6> 删除
    a. del
    删除指定位置的元素或删除整个列表
b. remove()
    删除首次出现的指定元素
c. pop()
    删除并返回指定位置上的元素
  5. 元组
    和列表的区别
    <1> 元组是不可变序列，元组中的数据一旦定义就不允许改变
    如果要改变
    e.g
        tuple1 = tuple2 + tuple3
        其实是创建了一个新元组，把两个元组内容放进去
<2> 优点
    元组的处理和访问速度比列表快
    元组的保护性强
    元组可以用作字典的键(不可变特性)
<3> 相互转换
    tuple(a_list)
    list(b_tuple)
  6. 字符串
    <1> 重复
    "Hello"*3
out: HelloHelloHello
  7. 集合
    <1> 定义
    是一组对象的组合，是一个无序排列的、不重复的数据组合体
    <2> set()
    可以把列表、元组、字符串等类型的数据转换成集合类型数据
    <3> frozenset()
    创建一个冻结的集合，该集合不能添加或删除任何元素，可以作为字典的key
  8. try…except
    <1> 格式
    try:
    #尝试执行下面代码
except 报错类型:
    #一旦有这种报错类型，就执行下面的代码
    <2> e.g
    try:
    age = int(input('请输入一个整数：'))
except ValueError:
    print('要输入整数噢')
