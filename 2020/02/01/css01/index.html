css01——CSS编写的位置

发表于 2020-02-01 | 更新于 2020-02-02 | 分类于 CSS | 评论数：

网页分为三个部分
结构(HTML)
表现(CSS)
行为(Javascript)
CSS
层叠样式表
网页实际上是一个多层的结构，通过CSS可以分别为网页的每一个层来设置样式
总之一句话，CSS用来设置网页中元素的样式




实体







1、内联样式
可以把css样式编写到元素的style属性里
将样式直接编写到style属性中，这种样式我们称为内联样式
内联样式只对当前元素中的内容起作用
内联样式属于结构与表现耦合，不方便后期的维护，不推荐使用
2、内部样式表
将样式写到head中的style标签里
然后通过CSS的选择器来选中元素并为其设置各种样式
可以同时为多个标签设置样式，并且修改时只需要修改一处
内部样式表更加方便对样式进行复用
问题：
内部样式表只能对一个网页起作用
它里面的样式不能跨页面复用
3、外部样式表
可以将CSS样式编写到外部的CSS文件中
通过link标签引入外部文件
只要想使用这些样式的网页都可以对其进行引用，提高复用性，实用性最高
将样式编写到外部的css文件中，可以使用到浏览器的缓存机制，从而加快网页的加载速度
